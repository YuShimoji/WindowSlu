# WindowSlu 開発申し送り

**最終更新**: 2024-12-01

## 現在の状態

### 完了済み機能（v2.1）

| 機能 | 状態 | 備考 |
|------|------|------|
| グループ化機能 | ✅ 完了 | プロセス名ベース自動グループ化 |
| プリセット機能 | ✅ 完了 | サイズ・透明度・位置設定の保存・適用 |
| カスケード配置 | ✅ 完了 | グループ内ウィンドウの整列配置 |
| 連動ドラッグ | ✅ 完了 | デフォルトOFF、相対位置維持 |
| グループ一括透明度 | ✅ 完了 | グループヘッダーのスライダーで一括変更 |
| グループ一括サイズ | ✅ 完了 | グループヘッダーで幅×高さ指定して一括適用 |

### 最近の修正（v2.0.1 → v2.1）

1. **連動ドラッグ初期状態の修正**
   - `WindowGroup.LinkedDragEnabled` デフォルトを `false` に変更
   - `WindowInfo.IncludeInLinkedDrag` デフォルトを `false` に変更
   - 意図しないウィンドウ連動を防止

2. **サイズスライダー挙動の修正**
   - `slider.IsMouseCaptureWithin` ガードを追加
   - バインド時の自動リサイズを防止

3. **グループ一括制御の追加**
   - グループヘッダーに透明度スライダー追加
   - グループヘッダーにサイズ入力欄 + Applyボタン追加
