# WindowSlu ドキュメントへようこそ！

WindowSlu は、Windows 上の各ウィンドウを透過表示させたり、効率的に管理したりするためのデスクトップアプリケーションです。

このドキュメントでは、WindowSlu のインストール方法、各機能の使い方、開発への貢献方法などを説明します。

## 主な機能

- ウィンドウの透過率調整（個別設定可能）
- デイモード／ナイトモード切り替え
- アプリケーション自体のリサイズ
- ウィンドウのどこからでもドラッグ移動
- その他、ウィンドウ管理を便利にする機能

## はじめに

まずは `インストールガイド` をご覧になり、WindowSlu をセットアップしてください。
